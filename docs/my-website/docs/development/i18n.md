## Usage

```typescript
import { useT } from "@shared/hooks/utils/useT";

function MyComponent() {
  const t = useT();
  
  return (
    <div>
      <h1>{t("welcome.title")}</h1>
      <p>{t("welcome.description")}</p>
    </div>
  );
}
```

## Translation Files

Located in `shared/i18n/locales/`:

```json
// en.json
{
  "welcome": {
    "title": "Welcome to AI Planner",
    "description": "Your personal planning assistant"
  }
}
```

## Adding Translations

1. Add key to all language files (`en.json`, `pl.json`, `de.json`)
2. Use `useT()` hook in components
3. Fallback language is Polish

## Parameterized Translations

```json
{
  "greeting": "Hello, {{name}}!"
}
```

```tsx
t("greeting", { name: user.name })
```

## Language Switching

```typescript
import { changeLanguage } from "i18next";

changeLanguage("pl"); // Switch to Polish
```
